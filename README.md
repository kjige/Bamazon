# Bamazon
Purchase items from the Bamazon inventory!

## Features
- Customer features:
  - Input product ID
  - Input quantity
  - Displays price of item purchased
  - Displays stock quantity before and after purchase
- Manager features:
  - View all products
  - View low inventory
  - Add more inventory to products
  - Add new products
